# 配信画面メモパッド (OBS Memo Pad)

OBSのブラウザソースとして画面上に透過表示させ、ダブルクリックで簡易的なメモ帳を自由に配置できるツールです。
配信中のタスク管理や、視聴者への簡易なテキスト表示に役立ちます。

## 特徴
- **OBSのブラウザソースに最適化**: 背景は完全な透過で、配信のレイアウトを邪魔しません。スクロールバーも非表示に設定されています。
- **直感的な操作**: 全て画面上のマウス操作（対話モード）で完結し、設定パネルなどは一切不要です。
- **データ永続化**: ブラウザの `localStorage` によってデータの保存が行われており、ページを再読み込み・OBSを再起動してもメモの内容と位置が復元されます。
- **完全単一ファイル**: 外部のJSライブラリやCSSフレームワークを一切使用せず、`index.html` のみで動きます。

## OBS ブラウザソースの設定手順

1. OBSの「ソース」の「＋」アイコンをクリックし、「ブラウザ」を追加します。
2. 作成する際の名前を「配信メモパッド」等にし、「OK」をクリックします。
3. **ローカルファイル** にチェックを入れ、`index.html` を指定します。
   （または GitHub Pages 等にデプロイし、そのURLを入力します）
4. **幅** をキャンバスの横幅（例: `1920`）、**高さ** をキャンバスの縦幅（例: `1080`）に設定します。
5. **「表示されていないときにソースをシャットダウン」** は自由ですが、リロードしても状態が戻るためオンでも問題ありません。
6. 設定ウィンドウを閉じます。

## 操作方法

OBSのソース一覧で追加したブラウザソースを **右クリック** し、**「対話」** を選択することで対話ウィンドウが開き、操作可能になります。

- **新規メモの作成**: 画面の何もない領域を **ダブルクリック** します。
- **メモの移動**: メモの上部余白（24px程度）を **ドラッグ** します。
- **テキスト編集**: メモのテキスト部分を **クリック** し、キーボードで入力します。
- **リサイズ変更**: メモの4つの角を **ドラッグ** して大きさを変更します。
- **メモの削除**: メモ上で **右クリック** し、表示される確認ダイアログで「OK」を押します。

## 便利な機能（カスタマイズ）

URLの末尾にパラメーターを追加することで、見た目をカスタマイズできます。

### マーカーの色変更 (`?color=`)
* **16進数カラーコードでの指定例**: `index.html?color=ffb6c1` （先頭の `#` は省略可能で自動補完されます）
* **英名カラーでの指定例**: `index.html?color=lightblue`
※ パラメーターを指定しない場合、デフォルトの「黄マーカー風 (`#fce883`)」が適用されます。

### ダークテーマの利用 (`?theme=`)
* **ダークテーマの適用**: `index.html?theme=dark` （背景やテキストが暗い色調になります。デフォルトはライトテーマです。）

※ 複数のパラメーターを組み合わせる場合は、`&` で繋ぎます。
例: `index.html?color=ffb6c1&theme=dark`

## 主な仕様および制約

- 同時に作成可能なメモの最大数は **10個** です。
- メモの最小サイズは `100x50` です。
- ブラウザ機能に基づくプレーンテキスト専用であり、画像の挿入や文字の装飾はできません。
- PC上でのマウス操作（Pointer Events）に焦点を当てており、モバイルのタッチ環境での利用は想定されていません。
